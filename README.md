# React Text Highlighter

A lightweight text highlighting application built with React.

## Features

* Highlight selected text in the browser
* Persistent highlights using Local Storage
* Remove highlights with a click
* Light and Dark mode support
* Responsive design for desktop and mobile
* Overlap-aware highlight management

## Tech Stack

* React
* JavaScript
* DOM Selection & Range APIs
* Local Storage

## Getting Started

```bash
npm install
npm run dev
```
